# 貿易融資業務真實性審查 — 財報技巧及案例分析

## 財報審查核心技巧分類

從貿融業務真實性審查角度，財報分析需要「四條線並行」：帳務異常、合同異常、貨物流轉異常、商業邏輯異常。

### 帳務表現紅旗指標

**應收帳款失速增長**

- 營收增長20%，但應收帳款同期增長50%以上，顯示新增營收大部分為賒銷且收現能力惡化。
- 坏帳計提比例波動異常：如貿易規模大增但壞帳撥備率反而下降，違背審慎原則。

**存貨與毛利率異常**

- 存貨周轉率與應收帳款周轉率同時異常下降，但營收卻大幅上升，暗示「做數」。
- 毛利率異常穩定（如多年維持10%固定回報），違背市場價格波動常理，屬於「固定收益」型融資性貿易特徵。

**預付／預收款異常波動**

- 預付帳款比例異常激增，意味企業超出常規業務邏輯墊付資金，很可能為上下游提供融資。
- 現金周轉周期特別短（如<10天），但聲稱有實體生產企業上下游，不符合常理，存在「空轉」嫌疑。

**現金流與利潤背離**

- 損益表顯示大量營收和利潤，但經營活動現金流量淨額為負或遠低於淨利潤，顯示收入可能虛掛在應收帳款或存貨。
- 資金流向與貨物交付不符：如合同顯示三方交易，但銀行流水只有固定金額在封閉圈內循環。

## 真實案例深度解析：通號 vs 永安案

### 案情背景

- **原告**：通號物資集團（央企）
- **被告**：平頂山永安運輸貿易公司
- **表面交易**：煤炭買賣，合同、發票、貨權轉移單、結算單全部齊全。
- **實質認定**：法院最終認定為「融資性貿易」，買賣合同無效，實為借貸關係。


### 財報層面破綻分析

| 財務指標 | 正常貿易特徵 | 本案異常表現 | 審查結論 |
| :-- | :-- | :-- | :-- |
| 收益模式 | 毛利率隨市場波動 | 合同明確約定年回報率12%（600萬÷5000萬） | 固定收益=融資性質 |
| 資金投入 | 各方按交易需要投入 | 只有通號初始6000萬在循環流轉，其他方無新投入 | 資金空轉 |
| 貨物控制 | 買方驗收、倉儲、運輸記錄 | 合同明確「乙方不對貨物數量及質量負責」 | 無真實貨權轉移 |
| 交易結構 | 線性供應鏈 | 四方封閉循環：甲→乙→丙→丁，資金反向流回 | 人為增加環節 |
| 單據完整度 | 齊全 | 齊全（合同、發票、貨權單、對賬函全有） | 單據齊全≠交易真實 |

### 法院認定關鍵證據

**合同條款自相矛盾**

- 框架協議寫明「資金循環制使用」「甲方同意籌集資金5000萬元用於合作項目」，明顯為融資安排而非貨物採購。
- 補充協議直接約定「年投資回報率不低於12%」，屬於典型的借貸利息條款。

**實際履行行為背離買賣本質**

- 雙方往來函件（詢證函、保證函、退款保證函）中反复提及「欠款」「借款」「遺留問題」，而非「貨款」「交貨」等買賣用語。
- 後續簽訂的「補救合同」明確寫「本合同旨在處理出賣人對永安公司的預付款債權」，證明真實目的為債權處理。

**監管機構介入定性**

- 國有企業監事會2016年發函認定為「融資性煤炭貿易業務，形成大額資金損失風險」。
- 國資委2017年通報明確指出「以貿易為名，實為出借資金、無商業實質」。

## 實操審查技巧與檢查清單

### 購銷合同審查（第一道防線）

**紅旗條款**

- 合同約定固定收益率、回報率、資金佔用費等非貿易用語。
- 條款明確某方「不對貨物質量／數量負責」，違背買賣合同基本義務。
- 大量補充協議、抽屜協議，與主合同實質內容衝突。

**前五大客戶/供應商穿透**

- 主要交易對手集中在經銷商／代理商，而非實際使用物資的生產型企業。
- 供應商目錄與銷售客戶名單出現相同或相似企業，形成封閉循環。


### 貨物流轉審查（第二道防線）

**實物控制檢查**

- 抽查無法提供原始倉儲單、運輸GPS記錄、驗收簽收單等支持文件。
- 倉儲公司、物流公司獨立核實時，出現「查無此貨」「倉單號不存在」。

**費用匹配度分析**

- 銷售費用規模與貿易規模不匹配：如營收暴增但銷售人員數量不變、運輸費／倉儲費未同步增長。
- 若聲稱有大量實體貨物流轉，但財報中運輸費、保險費、倉儲費異常低甚至為零。


### 資金流向穿透（第三道防線）

**銀行流水交叉驗證**

- 交易對手帳戶只有短暫收款即刻全數轉走，無正常營運收支 pattern。
- Backward tracing：追溯資金最終去向，看是否流向獨立第三方還是在關聯方內部兜圈。

**發票與報關數據核對**

- 增值稅發票金額、品名、數量與海關報關數據、稅務申報不符。
- 出口退稅記錄與聲稱的出口貿易規模嚴重不匹配。


### 商業邏輯合理性（第四道防線）

**定價機制檢查**

- 長期大額往來，但價格長期脫離市場價（如銅價市場波動±20%，合同價格卻不變）。
- 「高買低賣」：上游採購價高於下游銷售價，違背商業常識，除非有隱藏的融資利息補償。

**交易環節合理性**

- 人為增加中間環節：A、C都能直接供貨給終端，卻刻意讓B介入做「二道販子」，且B無任何增值服務（不加工、不倉儲、不配送）。
- 循環貿易：甲→乙→丙→甲，貨物／資金繞一圈回到起點，每一環都虛增營收。

## 你的「太後檢查」可以這樣落地

**第一步：快速篩查（1-2日）**

- 拉取疑似公司近三年應收帳款、存貨、預付款、營收、毛利率數據，計算周轉率及增長率，標記異常值≥30%波動的項目。
- 調取前五大客戶／供應商清單，用企查查／天眼查看股權關係，識別關聯方或同一實控人。

**第二步：合同文本審查（2-3日）**

- 抽查大額合同（如單筆≥500萬），重點看定價條款、收益約定、貨物責任條款，標記「固定回報」「資金循環」等敏感表述。
- 核對上下游合同簽訂日期、標的、單價，看是否「同日簽訂、只差價格」。

**第三步：第三方核實（3-5日）**

- 抽樣打電話給倉儲公司、物流公司、終端客戶，核實實際收發貨、庫存、付款情況（不要提前通知被查公司）。
- 向稅務局調取增值稅申報、出口退稅記錄，與公司提供的發票金額交叉比對。

**第四步：銀行流水分析（2-3日）**

- 調取疑似公司及其主要交易對手在你行的帳戶流水，追蹤大額資金（如≥100萬）最終去向，畫出資金流向圖。
- 計算「資金停留時間」：若收款後<3天即全額轉出，且轉入方又是上游供應商，屬於典型空轉特徵。

## 補充：常見虛假貿易識別指標速查表

| 識別維度 | 虛假貿易特徵 | 審查方法 |
| :-- | :-- | :-- |
| 毛利率 | 固定／異常穩定 | 與同行業可比企業對比，波動>5%視為正常 |
| 應收帳款 | 增速遠超營收增速 | 計算應收帳款周轉率，同比下降>20%危險 |
| 預付款 | 占總資產>15% | 詢問預付款具體對象和業務用途，難以解釋為紅旗 |
| 存貨 | 占營收比例>30%且周轉率<2次/年 | 驗證存貨的真實性和變現能力 |
| 現金流 | 經營淨現金流<淨利潤的50% | 收入高但現金進不來，虛假可能性大 |
| 交易結構 | 同一實控人下多家企業環型交易 | 追蹤股權關係，識別隱藏的關聯方 |
| 交易對手 | 經銷商／代理商而非終端用戶 | 前五大客戶穿透，確認最終購買方 |
| 資金流向 | 款進帳戶<3天即轉出 | 銀行流水Backward tracing，關注資金最終去向 |
| 費用水平 | 運輸費、倉儲費不符合貿易規模 | 與同行業對標，差異>50%可疑 |
| 報關數據 | 出口報關與國內銷售發票數量不符 | 調取海關數據交叉驗證 |


***

**結論**：單靠財報數字和完整單據無法判定貿易真實性。必須採用「四線並行」（帳務+合同+物流+資金流）的立體審查方法，才能有效識別融資性貿易和虛假交易。
